# Projeto To-do

Esse projeto foi baseado em uma ideia retirada do site FrontEnd Mentor. Meu objetivo com ele foi o de treinar JavaScript.

## Ideia do projeto

A ideia aqui é criar um site de lista de tarefas, onde você poderá adicionar nova tarefa, concluir tarefa, ver quantas tarefas faltam ser concluídas.

## Neste projeto foram usados:

- HTML
- CSS
- JavaScript

## O que revisei com esse projeto foi:

- HTML e CSS 
- Manipulação do Dom
- Funções
- Classes
- LocalStorage
- Manipulação do objeto data

## Como foi fazer o projeto

No início do projeto estava bem insatisfeito por não estar conseguindo usar classes para a criação da tarefa. Depois de estudar consegui fazer com que a tarefa criada seja baseada em uma classe de tarefa. Além disso, consegui utilizar o localStorage para armazenar as tarefas ainda não concluídas e exibi-las quando o usuário fecha e abre uma nova seção.

## O que falta

- Criar um feedback visual para tarefas atrasadas
- Criar o localStorage para as tarefas concluídas
